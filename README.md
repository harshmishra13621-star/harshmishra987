### 🚀 Projects: Turning Logic into Reality

Each repository on my profile represents a milestone in my learning. I don't just write code for the sake of completion; I write it to solve problems and understand systems.

#### **1. College Course Record Manager (CCRM) – Java Console App**
This is one of my most significant projects to date. It is a menu-driven application designed to handle the complexities of academic management. 
* **The Logic:** Built entirely in Java, it manages student data, course assignments, and grading systems.
* **The Learning:** This project was my deep dive into OOP. I had to ensure that the `Student` and `Course` classes interacted logically, using inheritance to handle different types of users and encapsulation to protect sensitive data. It taught me how to manage state in a console environment and the importance of error handling to prevent system crashes during user input.

#### **2. SQL Database Systems & Schema Design**
I have worked on several database projects that involved converting abstract ER diagrams into functional, high-performance SQL databases.
* **The Scope:** My projects cover various domains, including Library Management, E-commerce Inventories, and Student Information Systems.
* **The Technicality:** I focused heavily on schema design, ensuring proper normalization to reduce redundancy. I also implemented foreign key constraints to maintain referential integrity and utilized triggers to automate tasks like updating stock levels or logging system changes.

#### **3. Python Automation Scripts**
I am a big fan of "lazy" engineering—if a task is repetitive, I write a script to do it for me.
* **Applications:** I’ve developed tools to embed student details into PDF templates, summarize large CSV files of academic records, and automate file organization. 
* **The Value:** These projects highlight the power of Python for quick, effective problem-solving and have saved me (and my peers) hours of manual work.

#### **4. Responsive HTML/CSS Projects**
My journey began with a simple HTML resume page. Over time, that page evolved from a basic document into a structured, styled, and responsive personal site. This progression taught me the nuances of the DOM, the importance of "Mobile First" design, and how to use CSS to create a professional aesthetic without sacrificing performance.
